  例子：
   <!DOCTYPE html>
   <html>
    <head>
     <meta http-equiv="Content-Type" content="text/html;charset=utf-8">
     <title><test</title>
    </head>
    <body>
     <input id="input" value="This is a test">
    </body>
   </html>
   <script type="text/javascript">
    var isChromium = !!window.chrome;
    if(isChromium){
     document.getElementById("input").style.color = "red";
    }
   </script>
  在Chrome中打开这样的页面，input的文字是红色的；在FF中却不是。JavaScript Hacks其实是根据一些js条件的判断结果来确定用户所用的浏览 器是哪一种，然后根据要求规定样式。而且，js判断条件和浏览器无关，任何浏览器中都可以写这样的判断条件，不像其他的一些hack方式，一种浏览器只能识别几种 特殊的符号，以下JS Hacks就不再写例子了，大概用法就是这样。
